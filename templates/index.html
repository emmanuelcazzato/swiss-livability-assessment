Welcome to the Swiss Livability Assessment System

This system uses fuzzy logic and computing with words to transform quantitative environmental measurements into human-understandable livability assessments.

🎯 What We Assess

  • Acoustic Comfort: Based on WHO 2018 noise guidelines
  • Daylight Quality: Based on EN 17037 standards
  • View Quality: Sky visibility and greenery exposure
  • Location Convenience: Accessibility to amenities

📊 How It Works

  1. Input: Environmental measurements (noise, daylight, views, POIs)
  2. Fuzzification: Convert numbers to linguistic terms (e.g., "Quiet", "Bright")
  3. Inference: Apply 15 expert-defined fuzzy rules
  4. Output: Fuzzy Livability Index (0-100) with linguistic label

🏆 Livability Levels

Excellent (65-100)

Quiet environment, high daylight, good views

Good (45-75)

Balanced environmental quality

Fair (25-55)

Moderate quality, some improvements needed

Poor (0-35)

Significant environmental challenges

📚 About This Project

This is a proof-of-concept implementation of the Swiss Residential Perceived Livability Assessment Framework, developed as part of an Urban Computing seminar. The system processes data from the Swiss Dwellings v3.0.0 dataset (45,176 dwellings) and uses a Mamdani Fuzzy Inference System to compute livability scores that align with human perception.
